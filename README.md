# ServiceRequestSystem

Вебсистема реєстрації та супроводу сервісних заявок для систем відеоспостереження офісних приміщень.

## Опис

Проєкт реалізує прототип вебсистеми для обліку сервісних заявок, призначення сервісних інженерів, зміни статусів, додавання коментарів, фіксації результатів виконання, роботи з об'єктами обслуговування, пошуку, фільтрації та перегляду статистики.

## Технології

- C#
- ASP.NET Core MVC
- Entity Framework Core
- Microsoft SQL Server / LocalDB
- Razor Views
- HTML, CSS, Bootstrap

## Основні можливості

- авторизація користувачів;
- рольовий доступ;
- створення та перегляд сервісних заявок;
- призначення сервісного інженера;
- зміна статусу заявки;
- додавання коментарів;
- фіксація результату виконання;
- історія змін;
- робота з об'єктами обслуговування;
- пошук і фільтрація;
- статистика.

## Запуск проєкту

1. Відкрити проєкт у Visual Studio.
2. Перевірити рядок підключення до бази даних у файлі `appsettings.json`.
3. У Package Manager Console виконати:

```powershell
Update-Database
```
4. Запустити проєкт.
